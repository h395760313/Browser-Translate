## 1. 产品概述与目标

### 1.1 产品定位

一款轻量级的 Chrome 浏览器翻译插件，帮助用户在网页浏览过程中快速完成中英文互译，无需切换标签页或使用外部翻译工具。

### 1.2 核心目标

- **即时翻译**：选中任意单词或句子，0.5 秒内显示翻译结果
- **双向互译**：自动识别语言方向（中文↔英文），无需手动选择源/目标语言
- **语音朗读**：支持翻译内容的语音朗读，帮助用户学习发音
- **无干扰体验**：翻译框随选随现、用完即走，不影响正常浏览

### 1.3 核心价值

- 将翻译操作从「多步骤」压缩为「选中即翻」
- 减少上下文切换，提升阅读效率
- 辅助语言学习（翻译 + 朗读双重输入）

## 2. 用户场景与使用流程

### 2.1 核心用户场景

| 场景 | 用户行为 | 期望结果 |
|------|---------|---------|
| 阅读英文文章遇到生词 | 鼠标双击选中单词 | 立即显示中文翻译 |
| 阅读中文文章遇到英文术语 | 鼠标双击选中单词 | 立即显示英文翻译 |
| 理解长难句 | 鼠标拖拽选中整句 | 显示完整句子翻译 |
| 学习单词发音 | 看到翻译结果后 | 点击朗读按钮听发音 |
| 批量翻译多个词句 | 连续选中不同文本 | 每次都弹出翻译框 |
| 关闭/隐藏翻译框 | 不想看到翻译框了 | 点击空白处或 Esc 关闭 |

### 2.2 典型使用流程

```
用户浏览网页
    │
    ▼
选中文本（单词 or 句子）
    │
    ▼
弹出翻译气泡框（0.3-0.5s）
    │
    ├── 查看翻译结果
    │       │
    │       ├── [可选] 点击 🔊 朗读
    │       │
    │       └── [可选] 点击复制图标
    │
    ▼
点击空白处 / 选中其他文本 / Esc → 关闭翻译框
```

## 3. 功能规格详细说明

### 3.1 核心功能

#### F1：划词翻译（划选翻译）

- **触发方式**：用户在网页上用鼠标选中任意文本（单词或句子）
- **语言识别**：自动检测选中文本的源语言（中文或英文）
- **翻译方向**：
  - 源语言为中文 → 翻译为英文
  - 源语言为英文 → 翻译为中文
  - 非中英文文本 → 显示「暂不支持该语言」
- **结果展示**：在选中文本附近弹出翻译气泡框，显示翻译结果
- **最小选中文本**：1 个字符
- **最大选中文本**：500 个字符（超出截断并提示）

#### F2：语音朗读（TTS）

- **触发方式**：点击翻译框中的朗读按钮（🔊 图标）
- **朗读规则**：
  - 原文为单词（≤5 个词）→ 朗读原文发音
  - 原文为句子（>5 个词）→ 朗读翻译结果（中文原文朗读中文，英文原文朗读英文）
- **朗读引擎**：使用浏览器原生 Web Speech API（`speechSynthesis`）
- **发音人**：使用系统默认语音（中文使用 zh-CN，英文使用 en-US）
- **朗读控制**：朗读过程中再次点击按钮可停止

#### F3：复制翻译结果

- **触发方式**：点击翻译框中的复制按钮（📋 图标）
- **行为**：将翻译结果复制到剪贴板，显示「已复制」提示（1.5s 后消失）

### 3.2 增强功能

#### F4：自动弹出 / 手动切换

- **默认模式**：选中即弹（Auto Popup）
- **手动模式**：用户可在设置中切换为「选中后按快捷键弹出」
- **快捷键**：默认 `Alt + T`（可在设置中自定义）

#### F5：快捷键全局支持

- 支持在任意网页中使用划词翻译
- 支持全局快捷键唤起翻译框（对剪贴板内容进行翻译）

#### F6：翻译历史记录

- 自动记录最近 50 条翻译记录
- 通过插件 popup 页面查看历史
- 支持一键清空历史
- 单条记录可复制或重新翻译

#### F7：多翻译引擎支持

- **默认引擎**：Google Translate API（免费层）
- **备用引擎**：用户可在设置中切换为 DeepL / OpenAI 翻译（需自备 API Key）
- **引擎切换**：在设置页面中选择并配置

#### F8：设置面板

- 弹出模式切换（自动 / 手动）
- 快捷键配置
- 默认翻译语言偏好（可指定固定方向，如「始终翻译为中文」）
- 翻译引擎选择与 API Key 配置
- 朗读语速调节（0.5x - 2.0x）
- 主题切换（浅色 / 深色 / 跟随系统）
- 导出 / 导入设置

### 3.3 功能优先级

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0（必须） | F1 划词翻译 | 核心功能 |
| P0（必须） | F2 语音朗读 | 核心功能 |
| P1（重要） | F3 复制结果 | 高频辅助 |
| P1（重要） | F8 设置面板 | 配置能力 |
| P2（可选） | F4 手动模式 | 降低干扰 |
| P2（可选） | F5 快捷键 | 效率提升 |
| P2（可选） | F6 历史记录 | 辅助学习 |
| P3（可选） | F7 多引擎 | 高级用户 |

## 4. 界面交互设计

### 4.1 翻译气泡框

#### 布局结构

```
┌──────────────────────────────────┐
│  原文（英文原词 或 灰色小字）         │
│  ─────────────────────────────   │
│  翻译结果（大字号，主文本）           │
│                                  │
│  [🔊 朗读]          [📋 复制]      │
└──────────────────────────────────┘
```

#### 尺寸与定位

- **宽度**：280px（最大），最小适应内容
- **最大高度**：200px（超出滚动）
- **定位**：默认出现在选中文本的正下方；如果下方空间不足，出现在上方
- **距文本边缘**：8px 间距
- **圆角**：12px
- **阴影**：柔和投影（`0 4px 12px rgba(0,0,0,0.15)`）

#### 视觉风格

- **配色**：
  - 浅色主题：白色背景 `#FFFFFF`，深灰文字 `#333333`，边框 `#E5E5E5`
  - 深色主题：深灰背景 `#2D2D2D`，白色文字 `#F5F5F5`，边框 `#444444`
- **字体**：
  - 原文：13px，灰色 `#888888`
  - 翻译结果：16px，深色/亮色，font-weight 500
- **按钮**：
  - 圆角按钮，hover 时背景变浅灰
  - 点击时缩放 0.95 过渡动画

#### 状态

| 状态 | 视觉表现 |
|------|---------|
| 加载中 | 翻译框显示旋转 loading 图标 + 「翻译中...」 |
| 翻译成功 | 显示完整翻译结果 |
| 翻译失败 | 显示「翻译失败，请重试」+ 重试按钮 |
| 不支持语言 | 显示「暂不支持该语言」 |
| 朗读中 | 朗读按钮变为停止图标（🔊→⏹） |
| 已复制 | 复制按钮变为「✓ 已复制」（1.5s 后恢复） |

## 5. 技术考量

### 5.1 架构概览

```
┌─────────────┐     ┌──────────────┐     ┌────────────────┐
│ Content     │────▶│ Background   │────▶│ Translation    │
│ Script      │     │ Script       │     │ API           │
│ (划词检测)   │     │ (API 调用)   │     │ (Google/DeepL) │
└─────────────┘     └──────────────┘     └────────────────┘
       │                   │
       ▼                   ▼
┌─────────────┐     ┌──────────────┐
│ Popup       │     │ Storage      │
│ (历史/设置)  │     │ (chrome.storage) │
└─────────────┘     └──────────────┘
```

### 5.2 关键技术点

#### 划词检测（Content Script）

- 使用 `mouseup` 事件监听文本选中
- 通过 `window.getSelection()` 获取选中内容
- 使用 `Range` API 计算选中区域在页面中的坐标，精确定位翻译框位置
- 注入 CSS 样式表避免污染页面样式

#### 翻译 API

- **Google Translate API**（默认）：
  - 免费层有频率限制（每分钟 5 次），建议加缓存层
  - API 端点：`https://translate.googleapis.com/translate_a/single`
- **备选**：
  - DeepL API（付费，翻译质量更高）
  - OpenAI API（灵活，支持自定义 prompt）

#### 缓存策略

- 对相同文本的翻译结果缓存 30 分钟
- 使用 `chrome.storage.local` 存储缓存
- 缓存 Key：`hash(源语言 + 原文)` → `翻译结果`

#### 语音合成（Web Speech API）

- 使用 `window.speechSynthesis.speak()` 实现
- 选择语音：`speechSynthesis.getVoices()` 遍历匹配 `zh-CN` 或 `en-US`
- 语速控制：通过 ` utterance.rate` 参数（范围 0.1 - 10，默认 1.0）
- 处理 `voiceschanged` 事件确保语音列表加载完成

#### 数据持久化

- 使用 `chrome.storage.sync` 存储用户设置（支持多设备同步，限 100KB）
- 使用 `chrome.storage.local` 存储翻译历史和缓存（不限大小，建议定期清理）

### 5.3 权限需求

```json
{
  "permissions": [
    "activeTab",          // 仅访问当前活动标签页
    "storage"             // 读写本地存储
  ],
  "host_permissions": [
    "https://translate.googleapis.com/*",  // Google 翻译 API
    "https://*.deeplor.com/*"              // DeepL（如果启用）
  ]
}
```

### 5.4 性能目标

| 指标 | 目标 |
|------|------|
| 翻译气泡弹出延迟 | < 500ms（网络良好时） |
| 页面加载性能损耗 | < 50ms（content script 初始化） |
| 内存占用 | < 30MB（空闲状态） |
| TTS 响应时间 | < 200ms |
| 缓存命中率目标 | > 60%（重复翻译场景） |

### 5.5 兼容性

- **Chrome**：v88+（Chromium 内核）
- **Edge**：v88+（Chromium 内核，插件可直接移植）
- **不支持**：Firefox、Safari（可扩展，需单独适配）

## 6. 非功能性需求

### 6.1 性能

- 划词响应时间 < 500ms（含网络延迟）
- 插件总内存占用 < 50MB
- 不影响页面渲染性能（content script 使用 `requestIdleCallback` 初始化）
- 支持离线缓存：已翻译过的内容在网络断开时仍可查看

### 6.2 安全

- 不收集用户翻译内容，仅本地存储
- API Key 不暴露在前端，通过 background script 中转请求
- 不请求过多权限，最小权限原则
- 翻译内容不经过第三方日志服务

### 6.3 可用性

- 首次安装后无需配置即可使用（默认 Google 自动翻译）
- 支持键盘操作（Esc 关闭、快捷键触发）
- 翻译框在页面边缘时自动调整位置，不被截断
- 错误状态有明确提示和重试机制

### 6.4 可访问性

- 按钮有足够的点击区域（≥ 44×44px）
- 关键文本有足够的颜色对比度（WCAG AA 标准）
- 支持屏幕阅读器（ARIA label）

### 6.5 可维护性

- 模块化架构：Content Script / Background Script / Popup 分离
- 支持热更新（通过 Chrome Web Store 发布新版本）
- 预留插件扩展接口（未来可添加词典、语法检查等功能）
